**Editor**는 각 블로그의 API를 사용하여 블로그에 글을 작성하고 작성한 글을 관리하는 목적으로 만들고 있습니다.
- [Tumblr의 API](https://www.tumblr.com/docs/en/api/v2)

## Run (개발용)

```
$ npm install

# run app (development)
$ npm start
```

## 배포용 앱 생성

```
$ npm install

# 현재 OS 앱 생성
$ npm run dist
```
